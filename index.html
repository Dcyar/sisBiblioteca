¿Qué es Sisbiblioteca?.

Es un sistema web de gestión de una biblioteca, tiene múltiples funciones que harán fácil el manejo de los prestamos de libros, los usuarios, etc.

¿Que cosas puedo hacer con este sistema?

Con este sistema de bibliotecas podrás:

  • Gestionar usuarios.
  • Gestionar libros.
  • Gestionar autores.
  • Gestionar editoriales.
  • Gestionar reservas (se explica mas adelante).
  • Gestionar préstamos.

¿Por que "gestionar reservas"?

Una de las cualidades que tiene el sistema, es que es "web", los usuarios pueden entrar a la plataforma desde donde estén, vía Internet, por lo que se podrá hacer una reserva de un libro, el sistema le dará un código único con el cual podrá acercarse a la biblioteca y realizar el préstamo mostrando el código que se le dio, ademas hay que agregar que el código solo tiene valides por 2 días desde que se hizo la reserva.

¿Cómo esta hecho?

El sistema esta hecho con las siguientes tecnologías.

  • Python.
  • Django (framework de python).
  • Pillow (librería de django para gestión de imágenes).
  • HTML5.
  • CSS3.
  • Stylus (pre procesador de css).
  • JavaScript.
  • Jquery.
  • Virtualenv(entornos aislados de python).
